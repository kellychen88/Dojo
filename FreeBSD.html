FreeBSD/ppc Project

Introduction

The FreeBSD/ppc project pages contain information about the FreeBSD port to the PowerPC® architecture. As with the port itself, these pages are still a work in progress.

The FreeBSD/ppc port is still a Tier 2 platform. That means it is not being fully supported by our security officer, release engineers and toolchain maintainers.

Latest News 05 January, 2011: Support for the Sony Playstation 3 has been committed to SVN. 13 July, 2010: Support for the powerpc64 architecture added. 03 March, 2008: Support for Freescale® PowerQUICC III MPC85XX family system-on-chip development boards has been committed into CVS. 27 February, 2008: FreeBSD 7.0 is the first release to officially support the FreeBSD/ppc port.
